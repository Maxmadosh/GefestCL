Политика конфиденциальности GefestAR

Дата обновления: 1 июня 2026 г.

GefestAR предназначен для инженерного обследования площадок установки антенно-мачтовых сооружений и формирования PDF-отчётов.

Какие данные используются

Приложение может использовать:

  • геопозицию устройства для фиксации координат точки установки мачты;
  • камеру для AR-проекции, фотообзора и фото объектов обследования;
  • фотографии, выбранные пользователем, для добавления в отчёт;
  • данные, введённые пользователем: ID сайта, название объекта, контакт арендатора, телефон, комментарии, чек-листы и инженерное заключение;
  • локальные данные электропитания и картографические данные для расчёта расстояний.

Как используются данные

Данные используются только для функций приложения:

  • отображение AR-проекции;
  • формирование ситуационного плана;
  • расчёт ближайших объектов электропитания;
  • создание PDF-протокола обследования;
  • хранение истории отчётов на устройстве.

Хранение данных

Отчёты, фотографии и настройки сохраняются локально на устройстве пользователя. Пользователь может удалить сохранённые отчёты внутри приложения.

Передача данных

GefestAR не продаёт данные пользователей и не использует данные для рекламного трекинга.

Если пользователь самостоятельно отправляет PDF-отчёт через системное меню iOS, передача выполняется выбранным пользователем способом: почта, мессенджер, облачное хранилище или другой сервис.

Геопозиция

Геопозиция используется для координат обследования, GPS-привязки фотографий и формирования отчёта. Приложение не использует геопозицию для рекламного трекинга.

Камера и фото

Камера используется для AR-проекции мачты и фотофиксации объекта. Фото добавляются в отчёт только по действию пользователя.

Контакты

Приложение не получает доступ к адресной книге. Контактные данные арендатора или инженера вводятся вручную пользователем и используются только в отчёте.

Дети

Приложение предназначено для профессионального использования инженерами и не ориентировано на детей.

Изменения политики

Политика может обновляться при изменении функциональности приложения. Актуальная версия доступна по ссылке, указанной в App Store Connect.
